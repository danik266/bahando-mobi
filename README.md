# Bahandi Mobile

Expo React Native приложение для списаний Bahandi.

Общий backend:

```txt
http://46.101.134.38:4000/api
```

## Запуск

```bash
npm install
npm start
```

PIN для demo:

- сотрудники: `1111`
- проверяющий: `9999`

Настройки API лежат в `.env.example`. Локальный `.env` в git не коммитится.
